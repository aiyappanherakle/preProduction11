7 {_event_type} {_yes} public or realtime String and Integer 30 If this item will be scheduled to start on a specific future date enter realtime|YYYY-MM-DD HH:MM:SS as the value (realtime separated by a pipe delimiter following a future start date) otherwise if this item is a regular fixed price listing or auction style event which will start immediately enter public into this field.
8 {_bid_type} {_yes} regular or fixed String 10 If you want to sell this item by letting people bid and / or choose to buy this item via buy now (if you entered a buy now price in column 4) then it is recommended that you enter regular into this field otherwise if you wish to sell this item via buy now only (no bidding) enter fixed into this field.
9 {_category_id} {_yes} (i.e.: 100) Integer 10 In order to assign this item to a specific category in the marketplace you must already know the category id number of the category you wish to assign this item to.

13 Item State {_no} (i.e.: Ontario) String 50 The state or province of where the item is located. If no state is present, the state will be that of the person's session who is uploading the bulk csv file.
14 Item Zip / Postal code {_no} (i.e.: M3N2J9 or 90210) String and Integer 25 The zip or postal code of where the item is located. If no zip or postal code is present, the zip or postal code will be that of the person's session who is uploading the bulk csv file.
